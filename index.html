About Our Project

We are creating a virtual reality music visualizer for the Oculus Rift. This will be one of the first of its kind, and while there are some other music visualizers for the Rift out there, we have some interesting ideas that are going to make this one special.

We will take a music file, extract some information from it, such as the amplitudes of different frequencies in the waveform, and use that information to manipulate a 3D scene in some interesting ways.

Next

Our Plan

We plan to use OpenGL and the Oculus Rift SDK to render 3D graphics to the Rift. This will involve specifying the geometry of a dynamic 3D scene and continuously rendering that scene from the point of view of two virtual cameras that are eye-distance apart, which is what allows the wearer of the Rift to perceive 3D. The Oculus Rift SDK provides information about the orientation and position of the user's head, so the virtual cameras in the 3D scene can be repositioned and reoriented accordingly.

We will also implement a GUI for the application which will allow a user to select an mp3 file from his computer to play in the application.
